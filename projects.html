PIXEL EDITOR

GROUP PROJECT

In a group of 6, I was in charge of developing the drawing functionality within the project which includes drawing with a pen, brush, spray can etc. I came up with creative ways to incorporate these functions such as recursively filling in an area in a specific colour for the paint can functionality or randomly colouring in pixels in a given radius to create a spray affect. I also allowed the user to pick a colour they may have already drawn with directly from the canvas to be used again.

Skills: Java, Object Oriented Programming, Version Control (GitLab), Team Work
